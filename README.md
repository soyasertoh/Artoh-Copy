# Artoh Copy

Artoh Copy is a lightweight desktop tool for Windows (for now) designed to copy multiple files without having to access them one by one.

The application allows you to scan complex project structures, select relevant files using an interactive tree view, and copy the formatted content to the clipboard with a single click.

---

## Installation

1. Go to the [Releases](https://github.com/soyasertoh/Artoh-Copy/releases) tab of this repository.

2. Download the latest packaged installer (the `.msix` file).

3. Double-click the downloaded file and follow the Windows wizard's instructions to safely install the application on your system.
